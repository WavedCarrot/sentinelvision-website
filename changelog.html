Release History

SentinelVision Pro is actively maintained and improved. New releases are deployed via the in-app one-click updater.

v4.4.7 Latest Security April 2026
  • SEC License keys are now device-locked — the Pi serial number is cryptographically baked into the HMAC signature at generation time. A key issued for one Pi will not activate on any other device — the signature check fails immediately
  • SEC Keys cannot be transferred, shared or reused on different hardware. There is no way to strip or re-sign the device binding without the signing secret
  • NEW Device Serial shown on License page — your Pi serial is displayed with a one-click copy button. Send it to us when ordering so we can generate a key bound to your device
  • IMP All previously issued keys are invalidated by this change and must be reissued. Contact us with your device serial to receive a new bound key
v4.4.6 Latest Bug Fix April 2026
  • FIX License page no longer shows EXPIRED when expiring today — was incorrectly showing ❌ EXPIRED — feature disabled when the license expires later today but the exact timestamp has not yet passed. Now correctly shows ⚠️ Expires today with a Renew Now button
  • FIX Dashboard banner now says “expires today” instead of the confusing “expires in 0 days” wording
  • FIX License badge correctly shows Expired (red) vs Licensed (green) vs Expires Today based on the actual expiry timestamp, not just the day
  • IMP Features remain fully active until the exact expiry moment, then disable automatically on the next request — no manual restart needed
