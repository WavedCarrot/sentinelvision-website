Release History

SentinelVision Pro is actively maintained and improved. New releases are deployed via the in-app one-click updater.

v4.6.9 Latest Feature April 2026
  • FEATURE Dark / light mode toggle — navbar button switches between dark and light themes; preference is saved to browser storage and applied instantly without a page reload
  • FEATURE Custom usernames for admin and viewer accounts — set a custom username for each account in Settings; username is shown on the navbar and on the login page; each account can update their own username independently
  • FEATURE Audit log — Settings page shows the last 100 login, logout, and system-change events with timestamp, action type, username, and IP address
  • FEATURE Disk full Telegram alert — warning sent when disk usage reaches 85%; alert resets automatically once usage drops below 80%; checked every hour

  • UX Mobile-friendly layout — camera grids, event cards, and stat panels stack to a single column on tablets and phones; tables are horizontally scrollable; buttons are full-width on small screens (breakpoints at 768 px and 480 px)

v4.4.7 Security Security April 2026

  • NEW Device Serial shown on License page — your device serial is displayed with a one-click copy button. Send it to us when ordering so we can generate a key bound to your device
  • IMP All previously issued keys are invalidated by this change and must be reissued. Contact us with your device serial to receive a new bound key
